# E-commerce Sales Analysis using SQL

This is a beginner-friendly SQL project where I created a small e-commerce database and wrote queries to analyze sales performance.

## Project Overview

The database includes information about customers, products, orders, and order items.  
The goal of the project is to practice SQL by answering business-style questions using joins, grouping, and aggregate functions.

## Tables Used

- customers
- products
- orders
- order_items

## SQL Concepts Practiced

- Creating tables
- Inserting data
- Primary keys and foreign keys
- JOIN
- GROUP BY
- ORDER BY
- SUM
- AVG
- COUNT
- Subqueries

## Example Questions Answered

- What is the total revenue?
- Which product categories generate the most revenue?
- Which customers spent the most?
- Which products sold the most?
- How many orders came from each country?
- What is the average order value?

## Purpose

I created this project to practice SQL and improve my understanding of how data can be stored, connected, and analyzed in a relational database.
